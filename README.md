# Fusering

¡Hola! Este es el repositorio de mi proyecto de llavero personalizado para Hack Club.

<img src="https://cdn.hackclub.com/019f445e-7f6a-7eb7-b8db-042cf8e3b361/whatsapp_image_2026-07-08_at_7.40.39_pm.jpeg" alt="Mi Llavero 3D de Hack Club" width="500">

## Detalles del Proyecto
- **Herramientas usadas:** Figma (Diseño 2D) y GitHub Pages (Visualización).
- **Modelado:** Onshape
- **Restricciones de impresión:** 80 mm x 80 mm x 6 mm.

## Características del Diseño
- Base geométrica estilizada tipo banderín.
- Iconos vectoriales de herramientas (engranaje y llave).
- Agujero de 5mm integrado para la argolla.
